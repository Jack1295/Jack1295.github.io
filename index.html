CSS Selectors

Tag Name Description
p Paragraph using the 'p' selector in css followed by curly brackets will target all the Paragraph elements in a html file
Example: p{
font-family: serif;}		 this will change all the paragraph elements to serif font
a Anchor using the 'a' selector in css followed by curly brackets will target all the Anchor elements in a html file
Example: a{
color: green;}		 this will change all the anchor elements text to the color green
h1 - h6 Heading using the h1, h2, h3, h4, h5 or h6 selector in css followed by curly brackets will target all the Heading elements of their respective sizes in a html file
Example: h1{
font-weight: bold;}		 this will change the font weight of all the h1 elements to bold
# by ID when you need to target the id of an element instead of every element that shares the same tag you use a hashtag follwed by id name and curly brackets
Example: #example{
font-size: 18px;}		 this will change the font size of any element with an id of 'example' to 18 pixels
. by Class when you need to target the class of an element instead of every element that shares the same tag you use a period follwed by class name and curly brackets
Example: .example{
text-align: center;}		 this will change the text alignment of any element with a class of 'example' to centered
* All when using the * (asterisk) selector this targets all the elements on a page
Example: *{
border:solid black 1px;}		 this will wrap EVERY element on the page with a 1 pixel, solid, black border
img Images using the img selector followed by curly brackets will target all the images in a html file
Example: img{
width: 250px;}		 this will resize the width of every imamge on the page to 250 pixels
